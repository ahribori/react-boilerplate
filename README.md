# React-Redux-Express Boilerplate
<br/>

 1. Install libs
 ```javascript
 npm install
 ```
 <br/>

 2. build

 ```javascript
 npm run build
 ```
 <br/>

 3. startup (build required)
 ```javascript
 npm run boot
 ```
 - http://localhost:3000/

 <br/>

 4. run development server
 ```javascript
 npm run dev
 ```
 - http://localhost:4000/
